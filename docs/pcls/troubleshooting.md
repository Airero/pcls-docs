# 服务器连不上？

服务器直连地址已更换

对于使用IP直连的玩家，应该更换为 ddns.z0z0r4.top:34009 

使用 pcls.z0z0r4.top 的玩家不变（推荐

如果无法连接，先重试，然后按照流程检测

CMD命令 ping ddns.z0z0r4.top

检查网络延迟，高于150ms一般难以接受

如果网络好但还是无法链接：

1. 命令 ipconfig /flushdns 刷新DNS

2. 若为pcls.z0z0r4.top 换直连IP，不支持SRV

若为直连IP…你换个网络罢

以下可能为服务器故障

如果无法连接服务器，请先重试
